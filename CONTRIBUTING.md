## Ways to Help

### Report Bugs
1. Found a problem? Open Issue on GitHub
2. Describe what went wrong
3. How to make it happen again
4. What your system is (Windows/Mac, Python version, etc)

### Suggest Features
1. Open Discussion on GitHub
2. Explain your idea
3. Why it's useful

### Fix Code
1. Fork the repo
2. Make changes
3. Test thoroughly
4. Submit pull request with description

### Improve Docs
1. Found typo or unclear part?
2. Fix it
3. Submit pull request

### Share
1. Star the repo ‚≠ê
2. Tell your friends
3. Make videos/guides
4. Post on mod sites

## Code Rules

Python:
- Clear variable names
- Add comments for complex stuff
- Keep functions small

JSON:
- Proper indentation (2 spaces)
- Valid format
- No extra commas

Docs:
- Clear writing
- Examples
- Headers
- Links work

---

## Testing

Before submitting:
1. Test in game
2. Try all commands
3. Test with co-op
4. Test multiple characters
5. Works with/without sounds

---

## Reporting Bugs

Include:
- Title (clear, short)
- Description (what's wrong)
- Steps (how to repeat)
- Expected (what should happen)
- Actual (what does happen)
- System info (OS, Python, mods)
